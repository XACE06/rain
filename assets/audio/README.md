# Audio Sources

The rain ambience files in this folder are Mixkit free sound effects, downloaded for local playback in the editor.

- `heavy-rain-ambience.mp3`: Mixkit rain ambience preview, source page: https://mixkit.co/free-sound-effects/rain/
- `thunderstorm-rain.mp3`: Mixkit thunder/rain preview, source page: https://mixkit.co/free-sound-effects/rain/

Mixkit states that its sound effects are free to download and ready to use under the Mixkit License:
https://mixkit.co/license/
